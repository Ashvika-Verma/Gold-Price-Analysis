# 📈 Gold Price Analysis using Python

## 📌 Overview

This project performs Exploratory Data Analysis (EDA) and predictive analysis on historical gold price data from 1947 to 2026. Using Python, the project visualizes long-term price trends and compares actual values with model predictions to gain meaningful insights into gold price movements.

---

## 🚀 Features

- Historical Gold Price Analysis
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Trend Visualization
- Predictive Analysis
- Actual vs Predicted Model Comparison

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

---

## 📂 Dataset

Historical Gold Prices (1947–2026)
